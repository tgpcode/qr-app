# PayFlow - Premium Payment App

Ứng dụng thanh toán QR & Thẻ tín dụng với giao diện hiện đại.

## Tính năng
1. **Quét QR Đa Năng**: Hỗ trợ quét mã cho Ví Trả Sau và Thẻ Tín Dụng.
2. **Tạo QR Ví Trả Sau**: Chỉ hỗ trợ thanh toán Điện, Nước, Hóa đơn (theo yêu cầu).
3. **Thanh Toán Thẻ**: Giao diện nhập liệu Visa/Mastercard trực quan với hiệu ứng lật thẻ 3D.

## Cài đặt & Chạy

1. Cài đặt thư viện:
```bash
npm install
```

2. Chạy ứng dụng:
```bash
npm run dev
```

3. Mở trình duyệt tại địa chỉ được hiển thị (thường là `http://localhost:5173`)

## Công nghệ
- React + Vite
- Framer Motion (Animations)
- Lucide React (Icons)
- QRCode.react (QR Generation)
- Pure CSS Modules (Premium Styling)
